# What is Javascript
> JavaScript is the **programming language** of **HTML** and the **Web**.
```html
<!DOCTYPE html>
<html>
<body>

<button type="button"
onclick="document.getElementById('demo').innerHTML = Date()">
Click me to display Date and Time.</button>

<p id="demo"></p>

</body>
</html>
```
